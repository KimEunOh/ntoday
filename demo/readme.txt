해당 폴더는 더미데이터를 생성하고, 시각화 하는 내용을 다루고 있음. 
파견업체에서 기록한 근무 직원들의 근무 내용에 대한 데이터를 상정하고 생성하였음.

dummy_data.ipynb - 더미데이터를 생성하는 파일
final_schedule_data_with_metrics2.csv - 최종적으로 생성된 더미데이터

visual_app - dash를 이용한 웹 기반 시각화 코드 파일
 ㄴ main.py - 메인 페이지 및 구동 파일
 ㄴ data_processing.py - 데이터 로드 및 기본 전처리
 ㄴ __pycache__ - 캐싱 데이터 저장 폴더
 ㄴ assets - bootstarp 템플릿 및 로고 이미지 
 ㄴ page - 시각화 내용을 담은 각 페이지 코드
